This guide will help you create a basic map application with markers using React Google Maps.

## Basic Setup

<Steps>
  <Step title="Import the required components">
    Start by importing the core components from the library:

    ```tsx
    import {APIProvider, Map, AdvancedMarker} from '@vis.gl/react-google-maps';
    ```

    The three essential components are:
    - `APIProvider`: Loads the Google Maps JavaScript API and provides context
    - `Map`: Renders the map
    - `AdvancedMarker`: Places markers on the map (requires a Map ID)
  </Step>

  <Step title="Wrap your app with APIProvider">
    The `APIProvider` component must wrap all components that use Google Maps:

    ```tsx
    function App() {
      return (
        <APIProvider apiKey={"YOUR_API_KEY_HERE"}>
          {/* Map and other components go here */}
        </APIProvider>
      );
    }
    ```

    <Warning>
      Replace `YOUR_API_KEY_HERE` with your actual Google Maps API key. See the [Installation](/installation) guide for details.
    </Warning>
  </Step>

  <Step title="Add a Map component">
    Inside the `APIProvider`, add a `Map` component with basic configuration:

    ```tsx
    <Map
      defaultCenter={{lat: 22.54992, lng: 0}}
      defaultZoom={3}
      gestureHandling="greedy"
      disableDefaultUI={true}
    />
    ```

    Key props:
    - `defaultCenter`: Initial map center coordinates
    - `defaultZoom`: Initial zoom level (0-22)
    - `gestureHandling`: Control scroll/touch behavior
    - `disableDefaultUI`: Hide default map controls
  </Step>

  <Step title="Add markers to your map">
    Place `AdvancedMarker` components inside the `Map`:

    ```tsx
    <Map
      defaultCenter={{lat: 53.54992, lng: 10.00678}}
      defaultZoom={10}
      mapId="DEMO_MAP_ID"
    >
      <AdvancedMarker position={{lat: 53.54992, lng: 10.00678}} />
    </Map>
    ```

    <Note>
      `AdvancedMarker` requires a Map ID. You can create one in the [Google Cloud Console](https://console.cloud.google.com/google/maps-apis/studio/maps).
    </Note>
  </Step>
</Steps>

## Complete Example

Here's a minimal working application:

```tsx App.tsx
import React from 'react';
import {createRoot} from 'react-dom/client';
import {APIProvider, Map, AdvancedMarker} from '@vis.gl/react-google-maps';

function App() {
  const position = {lat: 53.54992, lng: 10.00678};

  return (
    <APIProvider apiKey={"YOUR_API_KEY_HERE"}>
      <Map 
        defaultCenter={position} 
        defaultZoom={10} 
        mapId="DEMO_MAP_ID"
      >
        <AdvancedMarker position={position} />
      </Map>
    </APIProvider>
  );
}

export default App;
```

## Add Styling

The Map component needs explicit dimensions to render properly:

```tsx
<Map
  style={{width: '100vw', height: '100vh'}}
  defaultCenter={{lat: 22.54992, lng: 0}}
  defaultZoom={3}
  gestureHandling="greedy"
  disableDefaultUI
/>
```

## Customizing Markers

You can customize markers using the `Pin` component:

```tsx
import {AdvancedMarker, Pin} from '@vis.gl/react-google-maps';

<AdvancedMarker position={{lat: 20, lng: 10}}>
  <Pin
    background="#22ccff"
    borderColor="#1e89a1"
    glyphColor="#0f677a"
  />
</AdvancedMarker>
```

Or use custom HTML content:

```tsx
<AdvancedMarker position={{lat: 30, lng: 10}}>
  <div
    style={{
      width: 16,
      height: 16,
      background: '#1dbe80',
      border: '2px solid #0e6443',
      borderRadius: '50%',
      transform: 'translate(-50%, -50%)'
    }}
  />
</AdvancedMarker>
```

## Using Standard Markers

For basic markers without a Map ID, use the `Marker` component:

```tsx
import {Marker} from '@vis.gl/react-google-maps';

<Marker
  position={{lat: 10, lng: 10}}
  title="Click me!"
  onClick={() => alert('Marker clicked!')}
/>
```

<Info>
  `Marker` uses the older `google.maps.Marker` class, while `AdvancedMarker` uses the newer Advanced Markers API with better performance and customization options.
</Info>

## Troubleshooting

<AccordionGroup>
  <Accordion title="Map not displaying">
    - Verify your API key is correct and the Maps JavaScript API is enabled
    - Ensure the Map component has explicit width and height styles
    - Check browser console for API errors or billing issues
  </Accordion>

  <Accordion title="AdvancedMarker not showing">
    - Make sure you've specified a valid `mapId` prop on the Map component
    - Create a Map ID in the [Google Cloud Console Maps Management](https://console.cloud.google.com/google/maps-apis/studio/maps)
    - Verify the marker position is within the map bounds
  </Accordion>

  <Accordion title="TypeScript errors">
    - The library includes TypeScript definitions automatically
    - Make sure you're using TypeScript 4.1 or higher
    - Check that `@types/google.maps` is installed (it's included as a dependency)
  </Accordion>

  <Accordion title="Map loads slowly">
    - The Maps JavaScript API is loaded dynamically on first render
    - Consider adding a loading state while the API initializes
    - Use the `onLoad` callback on APIProvider to detect when the API is ready
  </Accordion>
</AccordionGroup>
