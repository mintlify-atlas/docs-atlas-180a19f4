## What is React Google Maps?

React Google Maps is a TypeScript/JavaScript library that provides React components and hooks for seamlessly integrating the Google Maps JavaScript API into your React applications. Built by the vis.gl team, it offers a collection of components to create maps, markers, and info windows, along with hooks to use Google Maps services and libraries.

## Quick Example

Here's a minimal example to render a map with a marker:

```tsx
import {APIProvider, Map, AdvancedMarker} from '@vis.gl/react-google-maps';

function App() {
  const position = {lat: 53.54992, lng: 10.00678};

  return (
    <APIProvider apiKey={'YOUR_API_KEY'}>
      <Map 
        defaultCenter={position} 
        defaultZoom={10}
        mapId="DEMO_MAP_ID"
      >
        <AdvancedMarker position={position} />
      </Map>
    </APIProvider>
  );
}
```

## Design Philosophy

This library provides a **reactive wrapper** for the Google Maps JavaScript API while maintaining the map's ability to manage its own state. The components can work in:

- **Uncontrolled mode** - Map state is managed internally (default)
- **Controlled mode** - Full control over map state from React
- **Externally controlled mode** - Disable all built-in controls for external management

This flexibility allows you to integrate maps seamlessly into your React application architecture, whether you need simple map displays or complex synchronized multi-map interfaces.
