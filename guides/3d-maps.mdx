The `Map3D` component enables you to create immersive 3D map experiences with photorealistic 3D tiles, elevation data, and advanced camera controls.

## Basic 3D Map

Create a 3D map by using the `Map3D` component instead of the regular `Map` component:

```tsx
import { APIProvider, Map3D } from '@vis.gl/react-google-maps';

function My3DMap() {
  return (
    <APIProvider apiKey={API_KEY}>
      <Map3D
        defaultCenter={{ lat: 37.7749, lng: -122.4194, altitude: 1000 }}
        defaultRange={5000}
        defaultHeading={0}
        defaultTilt={45}
        mode="SATELLITE"
        style={{ width: '100vw', height: '100vh' }}
      />
    </APIProvider>
  );
}
```

## Camera Properties

The 3D map camera is controlled by these properties:

### Center with Altitude

The center position includes latitude, longitude, and altitude:

```tsx
<Map3D
  center={{
    lat: 37.72809,
    lng: -119.64473,
    altitude: 1300  // Meters above ground
  }}
  range={5000}
  heading={61}
  tilt={69}
  mode="SATELLITE"
/>
```

### Camera Parameters

- **center** - Position with latitude, longitude, and altitude (meters)
- **range** - Distance from camera to center point (meters)
- **heading** - Rotation in degrees (0-360, 0=North)
- **tilt** - Camera angle in degrees (0=straight down, 90=horizon)
- **roll** - Camera roll in degrees (typically 0)

<Note>
Use `defaultCenter`, `defaultRange`, etc. for uncontrolled mode, or `center`, `range`, etc. for controlled mode.
</Note>

## Map Modes

The `Map3D` component supports two rendering modes:

```tsx
import { Map3D, MapMode } from '@vis.gl/react-google-maps';

function SatelliteMap() {
  return (
    <Map3D
      mode={MapMode.SATELLITE}  // Photorealistic 3D tiles
      defaultCenter={{ lat: 37.7749, lng: -122.4194, altitude: 1000 }}
      defaultRange={5000}
    />
  );
}

function HybridMap() {
  return (
    <Map3D
      mode={MapMode.HYBRID}  // Satellite + street labels
      defaultCenter={{ lat: 37.7749, lng: -122.4194, altitude: 1000 }}
      defaultRange={5000}
    />
  );
}
```

### Available Modes

- `MapMode.SATELLITE` - Photorealistic satellite imagery without labels
- `MapMode.HYBRID` - Satellite imagery with street and place labels

## Controlled Camera

Manage camera state in your component:

```tsx
import { useState, useCallback } from 'react';
import { Map3D, Map3DCameraChangedEvent } from '@vis.gl/react-google-maps';

type CameraProps = {
  center: google.maps.LatLngAltitudeLiteral;
  range: number;
  heading: number;
  tilt: number;
  roll: number;
};

function Controlled3DMap() {
  const [viewProps, setViewProps] = useState<CameraProps>({
    center: { lat: 37.72809, lng: -119.64473, altitude: 1300 },
    range: 5000,
    heading: 61,
    tilt: 69,
    roll: 0
  });

  const handleCameraChange = useCallback((ev: Map3DCameraChangedEvent) => {
    setViewProps(oldProps => ({ ...oldProps, ...ev.detail }));
  }, []);

  return (
    <Map3D
      {...viewProps}
      onCameraChanged={handleCameraChange}
      mode="SATELLITE"
      style={{ width: '100vw', height: '100vh' }}
    />
  );
}
```

## Gesture Handling

Control how user gestures interact with the map:

```tsx
import { Map3D, GestureHandling } from '@vis.gl/react-google-maps';

<Map3D
  gestureHandling={GestureHandling.GREEDY}  // or COOPERATIVE, AUTO
  defaultCenter={{ lat: 37.7749, lng: -122.4194, altitude: 1000 }}
  defaultRange={5000}
  mode="SATELLITE"
/>
```

### Gesture Modes

- `GestureHandling.GREEDY` - Map captures all gestures (can't scroll page)
- `GestureHandling.COOPERATIVE` - Requires modifier keys or two fingers
- `GestureHandling.AUTO` - Map decides based on context (default)

## 3D Markers

Add markers to your 3D map using the `Marker3D` component:

```tsx
import { Map3D, Marker3D } from '@vis.gl/react-google-maps';

function MapWith3DMarkers() {
  return (
    <Map3D
      defaultCenter={{ lat: 37.7749, lng: -122.4194, altitude: 1000 }}
      defaultRange={5000}
      mode="SATELLITE"
    >
      <Marker3D
        position={{ lat: 37.7749, lng: -122.4194, altitude: 100 }}
        label="Marker 1"
      />
      <Marker3D
        position={{ lat: 37.7849, lng: -122.4094, altitude: 150 }}
        label="Marker 2"
      />
    </Map3D>
  );
}
```

<Note>
The `Marker3D` component requires an altitude value to position markers in 3D space.
</Note>

## Camera Animations

Animate the camera to fly to new locations:

```tsx
import { useRef } from 'react';
import { Map3D, Map3DRef } from '@vis.gl/react-google-maps';

function AnimatedMap() {
  const mapRef = useRef<Map3DRef>(null);

  const flyToLocation = () => {
    mapRef.current?.flyCameraTo({
      endCamera: {
        center: { lat: 40.7589, lng: -73.9851, altitude: 300 },
        range: 3000,
        heading: 45,
        tilt: 60
      },
      durationMillis: 3000
    });
  };

  const flyAroundPoint = () => {
    mapRef.current?.flyCameraAround({
      camera: {
        center: { lat: 37.7749, lng: -122.4194, altitude: 0 },
        range: 2000,
        tilt: 67.5
      },
      durationMillis: 20000,
      rounds: 1
    });
  };

  return (
    <>
      <button onClick={flyToLocation}>Fly to NYC</button>
      <button onClick={flyAroundPoint}>Orbit Location</button>
      
      <Map3D
        ref={mapRef}
        defaultCenter={{ lat: 37.7749, lng: -122.4194, altitude: 1000 }}
        defaultRange={5000}
        mode="SATELLITE"
      />
    </>
  );
}
```

### Animation Methods

- `flyCameraTo()` - Fly camera to a destination
- `flyCameraAround()` - Orbit around a center point
- `stopCameraAnimation()` - Stop ongoing animation

## Event Handling

Handle user interactions and camera changes:

```tsx
import { Map3D, Map3DCameraChangedEvent, MapMouseEvent } from '@vis.gl/react-google-maps';

function InteractiveMap() {
  const handleCameraChange = (event: Map3DCameraChangedEvent) => {
    console.log('Camera changed:', event.detail);
  };

  const handleClick = (event: MapMouseEvent) => {
    if (event.detail.latLng) {
      console.log('Clicked at:', event.detail.latLng);
    }
  };

  return (
    <Map3D
      defaultCenter={{ lat: 37.7749, lng: -122.4194, altitude: 1000 }}
      defaultRange={5000}
      mode="SATELLITE"
      onCameraChanged={handleCameraChange}
      onClick={handleClick}
    />
  );
}
```

## Mini Map Integration

Combine a 3D map with a 2D overview map:

```tsx
import { useState, useCallback } from 'react';
import { Map3D, Map, AdvancedMarker, MapMouseEvent } from '@vis.gl/react-google-maps';

function MapWithMinimap() {
  const [center, setCenter] = useState({ lat: 37.7749, lng: -122.4194, altitude: 1000 });

  const handleMiniMapClick = useCallback((event: MapMouseEvent) => {
    if (event.detail.latLng) {
      setCenter({
        ...event.detail.latLng,
        altitude: 1000
      });
    }
  }, []);

  return (
    <>
      {/* Main 3D Map */}
      <Map3D
        center={center}
        range={5000}
        mode="SATELLITE"
        style={{ width: '100vw', height: '100vh' }}
      />

      {/* Mini Overview Map */}
      <div style={{
        position: 'absolute',
        bottom: 20,
        right: 20,
        width: 200,
        height: 200,
        border: '2px solid white',
        borderRadius: 8
      }}>
        <Map
          center={{ lat: center.lat, lng: center.lng }}
          zoom={10}
          disableDefaultUI
          onClick={handleMiniMapClick}
        >
          <AdvancedMarker
            position={{ lat: center.lat, lng: center.lng }}
          />
        </Map>
      </div>
    </>
  );
}
```

## Disabling Labels

Hide default labels for a cleaner view:

```tsx
<Map3D
  defaultCenter={{ lat: 37.7749, lng: -122.4194, altitude: 1000 }}
  defaultRange={5000}
  mode="SATELLITE"
  defaultLabelsDisabled={true}
/>
```

## Photorealistic 3D Tiles

The `Map3D` component automatically renders photorealistic 3D tiles where available. These tiles provide:

- Accurate building geometries
- Realistic textures from aerial imagery
- Terrain elevation data
- Tree and vegetation rendering

<Note>
Photorealistic 3D tiles are available in select cities worldwide. In other areas, the map will show satellite imagery with elevation data.
</Note>

## Performance Considerations

<Warning>
3D maps are more resource-intensive than 2D maps. Consider these tips for optimal performance:
</Warning>

- Limit the number of 3D markers on screen
- Use appropriate `range` values (not too close or too far)
- Avoid rapid camera animation updates
- Test on mobile devices for performance
- Consider loading states for slower connections

## Complete Example

Here's a full-featured 3D map example:

```tsx
import { useState, useCallback, useRef } from 'react';
import {
  APIProvider,
  Map3D,
  Map3DRef,
  Marker3D,
  Map3DCameraChangedEvent,
  MapMode
} from '@vis.gl/react-google-maps';

type Location = {
  name: string;
  center: google.maps.LatLngAltitudeLiteral;
  range: number;
  heading: number;
  tilt: number;
};

const locations: Location[] = [
  {
    name: 'San Francisco',
    center: { lat: 37.7749, lng: -122.4194, altitude: 500 },
    range: 3000,
    heading: 0,
    tilt: 60
  },
  {
    name: 'New York',
    center: { lat: 40.7589, lng: -73.9851, altitude: 400 },
    range: 2500,
    heading: 45,
    tilt: 67
  }
];

function Full3DMapExample() {
  const mapRef = useRef<Map3DRef>(null);
  const [mode, setMode] = useState<MapMode>(MapMode.SATELLITE);
  const [cameraState, setCameraState] = useState(locations[0]);

  const handleLocationChange = (location: Location) => {
    mapRef.current?.flyCameraTo({
      endCamera: location,
      durationMillis: 3000
    });
  };

  const handleCameraChange = useCallback((event: Map3DCameraChangedEvent) => {
    setCameraState(prev => ({ ...prev, ...event.detail }));
  }, []);

  return (
    <APIProvider apiKey={API_KEY}>
      {/* Controls */}
      <div style={{
        position: 'absolute',
        top: 20,
        left: 20,
        zIndex: 1,
        background: 'white',
        padding: 16,
        borderRadius: 8,
        boxShadow: '0 2px 6px rgba(0,0,0,0.3)'
      }}>
        <h3>Locations</h3>
        {locations.map(loc => (
          <button
            key={loc.name}
            onClick={() => handleLocationChange(loc)}
            style={{ display: 'block', marginBottom: 8 }}
          >
            {loc.name}
          </button>
        ))}
        
        <h3>Mode</h3>
        <button onClick={() => setMode(MapMode.SATELLITE)}>
          Satellite
        </button>
        <button onClick={() => setMode(MapMode.HYBRID)}>
          Hybrid
        </button>

        <h3>Camera Info</h3>
        <p>Altitude: {cameraState.center?.altitude?.toFixed(0)}m</p>
        <p>Range: {cameraState.range?.toFixed(0)}m</p>
        <p>Heading: {cameraState.heading?.toFixed(0)}°</p>
        <p>Tilt: {cameraState.tilt?.toFixed(0)}°</p>
      </div>

      {/* 3D Map */}
      <Map3D
        ref={mapRef}
        {...cameraState}
        mode={mode}
        onCameraChanged={handleCameraChange}
        style={{ width: '100vw', height: '100vh' }}
      >
        {locations.map(loc => (
          <Marker3D
            key={loc.name}
            position={loc.center}
            label={loc.name}
          />
        ))}
      </Map3D>
    </APIProvider>
  );
}
```

## Best Practices

- Start with sensible default camera positions for your use case
- Provide UI controls for users to navigate and adjust the view
- Use `defaultLabelsDisabled` when you want a cleaner aesthetic
- Test camera ranges to find the sweet spot for your content
- Consider adding a 2D mini-map for context
- Provide fallback content for devices that don't support 3D
