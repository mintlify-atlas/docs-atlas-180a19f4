## Running Examples Locally

Each example is self-contained and can be run independently:

```bash
# Clone the repository
git clone https://github.com/visgl/react-google-maps.git
cd react-google-maps/examples/[example-name]

# Install dependencies
npm install

# Start the example
npm start
```

<Note>
All examples require a valid Google Maps API key. Set your API key in the `GOOGLE_MAPS_API_KEY` environment variable or directly in the code.
</Note>
